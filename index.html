An Ecosystem of Competing IONs

ARGOS is a "nervous system" of Intelligent Oracle Nodes (IONs) that stake influence to provisionally answer any query.

Evolve or Die

IONs that provide "truthful" data are rewarded, increasing their influence. Those that are successfully challenged ("The Fallen") are slashed and lose influence.

Cryptoeconomic Security

Anchored in UMA's optimistic oracle, the system is secured by cryptoeconomic incentives. The "cost of corruption" always outweighs the "profit from corruption".

Swarm Intelligence

Like a flock of birds or a colony of ants, the collective "swarm" of IONs produces an intelligent, adaptive, and self-healing data network far smarter than any single node.

Instant, Provisional Reality

Querying ARGOS is simple. Stake influence to back a provisional answer, which becomes final after the challenge period.

Other IONs are incentivized to challenge incorrect data, ensuring the "blockchain truth" is always found. This example shows a simple price feed query.

Quick Start Guide 

# 1. A query is submitted to the ARGOS network
async def get_price(asset, timestamp):
    log("Querying ARGOS for {asset}...")

    return await argos.query(
        f"price(asset={asset}, timestamp={timestamp})",

        # Stake 'influence' to back this provisional answer
        stake=1000,

        # Wait 1 hour for challenges
        challenge_period=3600
    )

# 2. Competing IONs battle to find the truth
# 3. 'Victors' are rewarded, 'The Fallen' are slashed
result = await get_price("ETH", 1678886400)
print(f"Provisional Price: {result}")
